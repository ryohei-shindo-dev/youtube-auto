# 正しいのに退屈で続かない。その感覚の整理

長期投資は正しいはずなのに、退屈で続きにくい人へ。正しいとわかっていても気持ちが離れやすい理由と、その感覚の整理方法をまとめます。

## 退屈さは失敗ではなく構造の一部

積み立て設定をして、毎月積み上がるのを待つ。これが長期投資向け投信の最も自然な形です。

やることが少ないほど「これで本当にいいのか」と感じやすくなります。勉強したのに使う場面がない。

ニュースを見ても何もしない。そうなると、正しい行動が急に味気なく見えてきます。

長期投資は構造上、途中の刺激が非常に少ないものです。毎日何かが起きるわけではなく、見た目は静かなままの期間が長く続きます。

静かすぎることが不安に見えてしまうだけのことが多くあります。退屈という感覚は、失敗のサインではなく自然に出てくるものです。

## 退屈さに対処する考え方

退屈になると、人は意味を足したくなります。別の商品を足す、売買の頻度を上げる、ニュースに合わせて動く。そうすると「投資している感じ」は強くなるでしょう。

しかしその手応えが、長期の成果と一致するとは限りません。静かすぎて信じにくいだけで、今のやり方が壊れているわけではないのです。

退屈な日ほど何かを足したくなります。その衝動が出るたびに「やり方が間違っている」と決める必要はありません。

今日は退屈だった、で終える。静かさをすぐに欠陥と結びつけないだけでも、気持ちは整いやすくなります。

過去の長期投資では、刺激の少ない期間を受け入れた人ほど、方針を保ちやすかった傾向があります。
