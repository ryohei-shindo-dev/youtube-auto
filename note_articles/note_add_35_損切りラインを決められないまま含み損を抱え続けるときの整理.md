# 損切りラインを決められないまま含み損を抱え続けるときの整理

**損切りすべきか、持ち続けるべきか。どちらにも決められないまま時間が過ぎている。

その状態を整理します。**
## 決められないのは、「どちらも後悔しそう」だから

損切りラインを決められない人の多くは、知識が足りないのではなく、「どちらの後悔が大きいか」を比べて動けなくなっています。

売ったあとに戻ったらつらい。
持ち続けてさらに下がってもつらい。
この二重の恐怖が、判断を先送りさせています。

## 損切りラインを決められないまま固まるのは、前提が混ざっているから

短期売買なら、損切りラインを先に決める考え方があります。
でもインデックスの積み立てでは、そもそもその前提が合わないことがあります。

長期で持つつもりなのに、下がった瞬間だけ短期の判断基準を持ち込む。
この前提の混ざり方が、決められなさを強くしていることがあります。

## 決められない日は、決めなくてもいい

どうしても判断がつかない日は、「今日は決めない」で終えても大丈夫です。

積み立て設定が動いているなら、何もしないこと自体が一つの判断です。
焦って動くより、動かない日のほうが結果的に正解だったということは少なくありません。
