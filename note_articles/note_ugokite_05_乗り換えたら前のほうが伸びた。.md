# 乗り換えたら前の銘柄のほうが伸びた｜投資先を変えて後悔するときの考え方

投資先を乗り換えたあと、前の銘柄のほうが伸びて後悔していないでしょうか。**この記事では、そうした後悔が生まれる心理的な理由と、気持ちの整理のしかたをまとめます。**

## 乗り換え後に「あっちが良かった」と見える理由

乗り換えを決める直前は、もともと持っていた商品への不安が強くなっています。「成長が遅い」という疑問が頭を占めているのです。

その時点では、新しい商品のほうが魅力的に見えます。情報を集めるほど良さばかりが目に入り、乗り換える決断に至ります。

しかし乗り換え後は心理的な状態が変わります。「選ぶ」というアクションが終わると、以前の商品への関心が薄れやすいのです。

別れた商品は「あの時点での値段」で記憶が固定されます。その後の値動きは無意識に忘れ、気づくと「あっちのほうが伸びてる」という感覚だけが残るのです。

## 後悔から抜け出す方法

乗り換えた商品で落ち着くには、**自動化された仕組みに完全に戻ること**が効果的です。

毎月の積み立ては変えず、引き落とし日もそのままにしておきます。比較の時間も決め、「見直しは◯年に1回」と固定することが大切です。

感情的な判断は自然に減ります。毎月淡々と積み立てられることで、「こちらも確実に増えている」という体感が戻ってきます。

数ヶ月から半年経つと、別の商品を確認したいという気持ちは薄れていきます。乗り換えたことをずっと後悔する必要はありません。

その選択を「今から確認する」に切り替え、仕組みは動かさないようにすると落ち着きやすくなります。
