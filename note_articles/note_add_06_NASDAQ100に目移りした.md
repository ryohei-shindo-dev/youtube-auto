# NASDAQ100が目につく夜、実は揺れているのは心

テック企業の急成長を見るたびに、今の投資先でいいのか不安になる。

長期投資を続けていると、こういう夜はありますよね。SNSでNASDAQ100の好調さを見かけたり、友人から「テック系に乗り換えた」という話を聞いたり。自分の投資ポートフォリオを眺めながら、「もっと攻めた方がいいのかな」と迷う。その時、実は数字の不安より先に、気持ちの不安が生まれているのかもしれません。

## データで見る、短期の上下と長期の傾向

過去20年間、毎年パフォーマンスで「最高」だった資産クラスはその次の年に最高にならないことが一般的です。好調な期間の直後は、往々にして調整局面が入る傾向があります。つまり、今「良い」と見える投資先が、明日も明後日も「最も良い」とは限らない、という整理ができます。

## 「今」と「ずっと」を別の軸で考えてみる

焦りの正体は、短期の見栄えと長期の結果を同じ目盛りで比べてしまうことです。NASDAQ100が好調な「この数ヶ月」を眺めると、確かに魅力的に見えます。ただ、長期投資の成功は、好調な時期を全て手に入れることではなく、下落局面を経ても続けられるかどうかで決まる傾向があります。「今、乗り遅れた」という感覚は、実は「短期の相対比較」に心が揺さぶられているだけかもしれません。

## 揺れを静めるために、できる小さなこと

このモヤモヤした夜は、無理に判断を下さない方が落ち着きやすいです。代わりに、現在の投資ルールを書き出してみると落ち着く傾向があります。「月2万円は積み立てる」「リバランスは年1回」など、決めたことをもう一度目で確認するだけで、感情的な判断から一呼吸置くことができます。SNSは明日でも見られますから、今夜はそっと閉じてしまっても大丈夫です。
