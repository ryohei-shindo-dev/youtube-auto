# NASDAQ100に乗り換えたくなる理由｜積み立て投資で目移りするときの整理

**NASDAQ100の好調さを見ると、「今の投資先でいいのか」と迷うことがあります。その迷いの正体と、冷静に整理する視点をまとめます。**

## 短期と長期を同じ軸で比べてしまう落とし穴

過去20年間、毎年パフォーマンスで「最高」だった資産クラスが、翌年も最高になることはほとんどありません。好調な期間の直後には、調整局面が入るのです。

「今、良い」と見える投資先が、明日も「最も良い」とは限りません。焦りの正体は、短期の見栄えと長期の結果を同じ目盛りで比べてしまうことです。

NASDAQ100が好調な「この数ヶ月」を眺めると、確かに魅力的に見えます。ただ、長期投資の成功は、好調な時期を全て手に入れることではありません。下落局面を経ても続けられるかどうかで決まる傾向があります。

「今、乗り遅れた」という感覚は、実は「短期の相対比較」に心が揺さぶられているだけかもしれません。
