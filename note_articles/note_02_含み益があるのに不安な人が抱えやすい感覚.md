# 含み益があるのに不安な夜に起きやすいこと

**含み益があるのに不安で落ち着かない経験は珍しくありません。この記事では、利益が出ているのに安心できない理由と向き合う方法をお伝えします。**

## 不安が生まれる理由

ポートフォリオを開くとプラスの数字が並んでいるのに、ほっとしない人は多いものです。「このまま下がったらどうしよう」という気持ちが頭をよぎります。

行動経済学では、利益が出ている状態のほうが損失への感受性が高まると指摘されています。これは「プロスペクト理論」として知られています。

「今あるプラスを失いたくない」という心理が、不安を増幅させやすいのです。含み益がある状態は、失うものができた状態でもあります。感情が揺れやすくなるのは自然な反応と言えます。

## 不安な気持ちとの向き合い方

含み益のタイミングで感じる不安は、早めに利確しようという行動につながりやすいものです。しかし長期投資のデータを見ると、利確のタイミングを繰り返すことで長期的なリターンが小さくなるケースが多く観察されています。

不安を感じたときは、まずポートフォリオを開く回数を減らしてみましょう。感情の揺れが落ち着きやすくなります。

次に、「今の積立設定を変えない」を行動の基準として持つと判断しやすくなります。不安を感じた日は、「感情が動いた」と記録だけしておくことをお勧めします。それだけで衝動的な判断を一歩遠ざけられる傾向があります。

含み益があるのに不安なのは、投資がうまくいっている証拠でもあります。その感覚を「異常」と思わなくても大丈夫です。
