# 「他の投資の方が伸びてる」という夜の不安が、実は当たり前の理由

SNSで見かけた銘柄がストップ高。自分のインデックスファンドは淡々と積み立てるだけ。その差を目にした夜、方針が弱く思えてしまいます。

長期投資を続けている人なら、誰もが経験する感覚ですよね。特に相場が活発な時期には、「もっと伸びる何かがあるんじゃないか」という声がSNSに溢れます。その時、自分の地道な積み立てが、なんだか生ぬるく感じるかもしれません。でも、その感覚は実は、仕組みの問題かもしれません。

過去30年間の日本株式市場を見ると、わずか数銘柄の上昇がインデックス全体の上昇の30〜50%を占めていたとする分析があります。つまり、大きく伸びた銘柄の多くは、市場全体で見ると少数派なのです。

ここが大事な点です。「他の投資の方が伸びてる」という印象は、数百万人の失敗よりも、その1人の大成功がSNSで拡大されるからです。インデックス投資は、成功も失敗も地味です。だから目立たないだけで、統計的には多くの人がそこで結果を出しています。

この感覚に襲われたら、自分のポートフォリオの推移グラフを1年単位で眺めることをお勧めします。月単位の変動は心を揺さぶりますが、1年単位で見ると、淡々とした上昇が見えやすくなります。その上昇は、「少数派の大勝利」ではなく、「多数派の着実な成功」の積み重ねです。

不安な夜は、その1年グラフを見て、自分の方針がどこにいるかを確認するといいでしょう。

## よくある質問

**Q：本当に少数銘柄で市場全体が動いているんですか？**
A：過去のデータではそのような傾向が観察されています。ただし市場環境によって異なるため、一概には言えません。

**Q：じゃあインデックスでいい成績を出すのは難しいんですか？**
A：むしろ逆で、市場全体に分散投資することで、その少数派の銘柄の上昇をキャッチできる可能性があります。

**Q：目立つ銘柄を見てしまうのは、やっぱり投資家失格ですか？**
A：そうではありません。目立つものを見るのは人間の本能です。大事なのは、それに反応してから「冷静に戻る仕組み」を作ることです。
