# SNSで他人の爆益を見ると焦る理由｜積み立て投資で比較がつらいときに読む話

**SNSで他人の爆益投稿を見るたびに焦りを感じていませんか。積み立て投資を続けている人がSNSの比較で不安になるのは自然なことです。**

## SNSに映るのは「勝った人」だけ

「+300万円」「今月だけで2倍」——こんな投稿がタイムラインに流れてくる夜があります。地道に積立を続けているのに、自分だけ取り残されているような気持ちになる人も多いでしょう。

実は投稿されるのは「勝った人」だけです。負けた人、退場した人はほぼ投稿されません。

台湾の株式市場を対象とした学術研究では、個人デイトレーダーのうち継続的に利益を出せたのは約1%に過ぎません。SNSのタイムラインは「生存者だけの展示場」と言えるのです。

人は損失よりも「他者の利益」に強く反応する傾向があります。SNSの構造はこの傾向をさらに強めてしまいます。

## 見えていない損失とストレス

爆益スクショの背後には、見えていない損失・時間・ストレスがあります。1回の大きな利益の前後に何度の損失があったか、投稿者本人にしかわかりません。

一方、長期のインデックス投資は過去のデータに「地味に右肩上がり」という傾向が見られます。目立たないから投稿されにくいだけで、実は着実に積み上がっているものが手元にある可能性があります。

比較するなら「SNSの誰か」ではなく「1年前の自分の残高」と比較する方が、実態に近い評価ができます。

## 焦りを感じたときにできること

SNSを見る時間を、今日だけ30分減らしてみてください。落ち着きやすくなります。アプリの通知をオフにする、スクリーンタイムに上限を設けるなど、どちらでも構いません。

焦りを感じているときの判断は、傾向として後悔につながりやすいとされています。積立の設定は、今夜は触らないことをおすすめします。

積立を切らずに今夜をやり過ごすことが、長期投資家の共通点です。
