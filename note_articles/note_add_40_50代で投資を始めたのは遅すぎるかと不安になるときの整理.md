# 50代で投資を始めたのは遅すぎるかと不安になるときの整理

**50代で投資を始めた。でも「もう遅いのでは」という不安が消えない。

その気持ちを整理します。**
## 「遅い」と感じるのは、他人の開始年齢と比べているから

50代で始めたことを「遅い」と感じるのは、20代や30代で始めた人の話を見るからです。
SNSや書籍では、早く始めた人の成功例が目立ちます。

でも、投資を始めた時点で「始めていない人」よりは前に進んでいます。
比較の相手を「もっと早く始めた人」から「まだ始めていない自分」に変えると、見え方が変わります。

## 50代からでも、時間はゼロではない

50代で始めても、65歳まで15年、70歳まで20年あります。
20代や30代より短いとしても、時間がまったく残っていないわけではありません。

もちろん、早く始めた人と同じ条件にはなりません。
でも「遅いから意味がない」と「遅いけれど意味はある」は、かなり違う受け止め方です。

## 遅く始めた人にとって大事なのは、金額より「止めないこと」

50代からの投資で一番怖いのは、焦って大きく入れてすぐ不安になることです。

金額を増やすより、月々の積み立てを無理のない範囲で続けるほうが、気持ちが崩れにくいです。
遅く始めた分を取り返そうとして大きく動くと、逆に続けにくくなります。

始めた時期は変えられません。でも「続ける期間」はこれから作れます。
