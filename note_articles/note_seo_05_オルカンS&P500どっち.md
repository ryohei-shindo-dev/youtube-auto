# オルカンかS&P500か迷って決められない｜比較疲れの整理

オルカンかS&P500か迷って決められない人へ。この記事では、比較が止まらなくなる理由と、どちらが正解かではなく続けやすさから整理する考え方をまとめます。

オルカン（全世界株式）かS&P500（米国株式）か。どちらを選んでもいいはずなのに、一度比較し始めると、その判断からなかなか抜け出せなくなります。過去のリターン、地政学的リスク、ドルコスト平均法との相性…。調べれば調べるほど、正解から遠ざかっているような気がして、選択肢を増やしてしまう人も少なくありません。

その疲れは、「どちらが正解か」という問い方をしているからかもしれません。

## 比較疲れが強くなる理由

オルカンとS&P500を迷う心理には、誰にでも起きやすいパターンがあります。

まず、両者のリターン差がはっきり見えやすい点です。ここ数年、米国株の好調さが目立っていたため、「S&P500を選んだ方が増える？」と感じる人が多くいます。一方で「でも全世界に分散した方が安心？」という感覚も同時に生まれます。この二律背反が比較のスイッチを入れてしまうのです。

次に、「正解があるはず」という前提があることです。投資商品の比較は、どちらかが明らかに優れているわけではありません。なのに、ネット上には「オルカンが買い」「いやS&P500だ」という主張が相互に存在するため、どちらかを選べば「大丈夫」と思えるまで、判断を後回しにしてしまうのです。

この状態が、比較疲れの正体です。

## 迷いを整理する視点

大切なのは「どちらが正解か」ではなく「自分の条件で、どちらが続けやすいか」です。

続けやすさは人によって異なります。米国経済の先行きを信じる気持ちが強い人もいれば、複数国への分散を心理的に重要だと感じる人もいます。長期で積み立てる前提なら、選択そのものより続けやすさの影響が大きいと整理しやすいです。つまり、「どちらが正解か」より「その選択を続けられるかどうか」が、ずっと大切なのです。

迷ったときに役立つ確認は、こうしたものです。

- 日々のニュースで米国経済の話を聞いたとき、その見方についていけるか
- 逆に「全世界に賭ける」という安定感が必要か
- 口数や金額で比較していないか（買いやすさで選ぶと後悔しやすい）

この3つで整理すると、自分の心が本当に求めているものが見えやすくなります。

## 決めきれない日の終わり方

もし今日も決められなかったなら、その日に決めきれないなら、今日は選ばないと決める方が落ち着くこともあります。判断を急がず、まず比較の軸を書き出してみる方法もあります。短期間で結論を出さなくてもよい場面はあります。
