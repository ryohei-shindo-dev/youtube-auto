# 投資信託を増やしすぎると不安になる理由｜商品数を減らすと気持ちが整う話

**投資信託の保有本数が多すぎると、分散しているはずなのに逆に不安が増えることがあります。どれが何の役割なのか分からなくなり、安心感よりも迷いが大きくなるのです。**

SNSで他の投資家の話を聞くと、自分の選択に疑問が生じやすくなります。「あの人は5本、この人は20本持ってる」という情報に触れると、自分のポートフォリオが本当に最適なのか判断しにくくなるのです。

## 商品数を減らして気持ちを整える

行動経済学の研究では、選択肢が増えるほど意思決定に時間がかかり、決定後の満足度が低下する傾向があります。投資信託の場合、10本を超えると各銘柄の役割が不明確になりやすいです。

「本当に全部必要なのか」という疑問が日々増していくでしょう。管理の負担も心理的な負担も、比例して大きくなります。

過去のデータでは、投資初心者は3～5本程度の銘柄で十分な分散効果を得られるとされています。本数ではなく、納得度の高さが大切なのです。

長期投資で大切なのは、銘柄数ではなく続けることです。自分が説明できる本数に絞ると、月々の積み立て時も判断がしやすくなります。

相場が乱れた時も冷静に対応できます。「もっと増やさなければ」という不安ではなく、「今持っているもので大丈夫」と心が落ち着く構成を意識してみてください。

現在、投資信託を10本以上持っているなら、一度ポートフォリオを紙に書き出してみることをお勧めします。各銘柄の役割を説明できるか確認してください。

説明できないなら、それは「迷いの原因」かもしれません。必要に応じてシンプルな構成に整理し直すと落ち着きやすくなります。

売却時の税金や既に発生している損益を確認したうえで、無理のない範囲で進めることが大切です。すべて一度に変更する必要はありません。

納得度が高い方が、長期投資は続きやすいのです。
