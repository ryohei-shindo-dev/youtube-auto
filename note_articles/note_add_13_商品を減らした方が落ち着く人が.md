# 投資商品を減らしたら、なぜか心が落ち着いた話

投資商品を減らしたら心が落ち着いた経験がある人へ。この記事では、商品数が増えるほど迷いも増えやすい理由と、減らすことで判断が軽くなる感覚を整理します。

## 投資商品を増やしすぎると迷いやすくなる理由

長期投資をしていると、気づけば投資先が増えていることがあります。インデックスファンド数本に始まり、新しい商品が出るたびに「これも良さそう」と追加して。気がつくと10種類以上になっていたり。「たくさん分散させたら安心」という思い込みが、いつの間にか判断のストレスに変わっていることもあります。

## 投資商品を減らすと認知負荷が下がる理由

2023年に行われた消費者心理の研究では、選択肢の数が増えるほど決定時の精神的負担が増加する傾向が報告されています。投資商品もこれに該当するもので、管理する商品が多いほど「各商品の値動きを気にする」「本当にこれでいいのか迷う」という心理状態が続きやすいとされています。

実は、私たちの脳は「管理するもの」が増えるだけで疲れるようにできています。投資商品も、見守るべきものとして無意識に脳はリソースを使い続けているのです。

## 投資商品を減らすと心が落ち着きやすい理由

商品数を減らすと落ち着く理由は、判断の負担が減るからだと整理できます。毎日のポートフォリオ確認時も、迷う対象が少ないほど思考が単純になります。「この3本の積み立てを続ける」と決めた方が「この8本で本当にいいのか」と悩み続けるより、精神的なエネルギーが温存される傾向があります。

長期投資で大事なのは「最適なポートフォリオ」ではなく「続けられるポートフォリオ」です。複雑さが手放されることで、逆に続ける意志が強くなる可能性があります。

## 投資商品を減らしたいときにできること

もし今、投資商品の数に引っかかっているなら、一度ポートフォリオを眺めて「本当に必要な商品」と「なんとなく持っている商品」を仕分けしてみてはいかがでしょうか。減らすことをハードルと捉えず「心の解放」という視点で考えると、判断しやすくなることもあります。完璧を求めるより、シンプルに続く状態を整える方が、長期投資では強みになります。

シンプルなポートフォリオほど、長く続ける力を持っています。
