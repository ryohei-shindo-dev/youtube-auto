# 配当利回り5%の誘いに、過去データを確認してから返事する

目の前の高い配当利回りが、ずっと続くわけではないという不安。

インデックス投資を続けていると、SNSやYouTubeで「高配当銘柄なら毎年5%のリターンが見込める」といった投稿を目にします。一方で、自分が保有しているインデックスファンドの配当利回りは1～2%程度。その差を見ると、「今乗り換えたら、もっと豊かになれるのではないか」と心が揺らぐのは自然なことです。配当金として手に入るお金は目に見えやすく、心理的な満足感も大きいため、余計に魅力的に映るのでしょう。

配当利回りと株価上昇率を合わせた総リターンで見ると、過去30年間の米国株式市場では年平均約10%のリターンが得られていたとされます。この中には配当利回り（約2～3%）と株価上昇（約7～8%）の両方が含まれている形です。つまり、配当を多く出す企業は、株価の上昇幅が相対的に小さくなる傾向があります。

配当金を手にすることは気持ちの良いものですが、総リターンで見ると異なる構図が見えます。高配当戦略に乗り換えることで、手取りの配当は増えても、株価上昇の恩恵が減る可能性があります。つまり、「今の配当が少ないから乗り換える」という判断は、全体像を見落としているかもしれません。

高配当への乗り換えを検討する際は、配当利回りだけでなく、過去の総リターン（配当込み）を自分で調べてから判断すると落ち着きやすいです。手数料の違いや税制上の扱いなども、あわせて整理するとより冷静に考えられます。すぐに判断を急がず、1週間かけてゆっくり比較する時間をとってみてください。

総リターンと配当利回りは別の数字であり、どちらか片方だけを見ると判断を誤る可能性があります。

## よくある質問

**配当利回りが高い銘柄ほど儲かるのではないですか？**

配当利回りが高いことと総リターン（利益）が高いことは別です。配当を多く出す企業は成長が緩やかなため、株価の上昇幅は相対的に小さくなる傾向があります。

**現在の配当利回り2%は低いのでは？**

インデックスファンドに含まれる企業全体の利益の一部が配当として出され、残りは企業の成長に使われています。配当利回りが低く見えるのは、企業が成長に投資している側面もあります。

**高配当投資から乗り換えても大丈夫ですか？**

乗り換え時に売却益に税金がかかったり、売却後のタイミングによって成果が変わったりします。判断の前に、切り替えにかかる実際の費用と時間をシミュレーションすると落ち着きやすいです。
