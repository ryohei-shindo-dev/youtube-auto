# 配当利回り5%は本当に魅力的か｜高配当に乗り換えたくなったときの確認点

**配当利回り5%の高配当銘柄が気になったとき、乗り換えるべきか迷う人は少なくありません。配当利回りだけで判断すると見落としやすいポイントを、過去データをもとに整理します。**

## 配当利回りと総リターンは別の数字

インデックス投資を続けていると、「高配当銘柄なら毎年5%のリターンが見込める」という投稿を目にします。一方で、自分が保有しているインデックスファンドの配当利回りは1～2%程度です。

その差を見ると、乗り換えたくなるのは自然なことです。配当金は目に見えやすく、心理的な満足感も大きいため余計に魅力的に映ります。

過去30年間の米国株式市場では、年平均約10%のリターンが得られていました。この中には配当利回り（約2～3%）と株価上昇（約7～8%）の両方が含まれています。

配当を多く出す企業は、株価の上昇幅が相対的に小さくなる傾向があります。高配当戦略に乗り換えると、手取りの配当は増えても、株価上昇の恩恵が減る可能性があるのです。

乗り換えを検討する際は、配当利回りだけでなく、過去の総リターン（配当込み）を自分で調べてから判断してください。手数料の違いや税制上の扱いなども、あわせて整理すると冷静に考えられます。

乗り換え時に売却益に税金がかかったり、売却後のタイミングで成果が変わったりするため、実際の費用をシミュレーションしてみてください。すぐに判断を急がず、1週間かけてゆっくり比較する時間をとってみてください。
