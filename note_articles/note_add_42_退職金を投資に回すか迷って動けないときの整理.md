# 退職金を投資に回すか迷って動けないときの整理

**退職金が入った。投資に回したほうがいいのはわかる。

でも、一括で入れるのが怖くて動けない。その迷いを整理します。

## 動けないのは、金額が大きすぎて「失敗が許されない」と感じるから

退職金は、多くの人にとって人生で最大級のまとまった資金です。
これを投資に回すとなると、「間違えたら取り返しがつかない」という恐怖が生まれます。

月3万の積み立てなら許容できるリスクでも、退職金の全額となると話が変わります。
金額の大きさが、判断の重さをそのまま増幅しています。

## 「全額一括」か「全額貯金」の二択にしなくていい

迷いが生まれやすいのは、「投資に全額入れるか、全額置いておくか」の二択で考えてしまうからです。

実際には、一部だけ投資に回して残りは預金に置く考え方もあります。
ある程度を手元に残し、残りを時間をかけて移していく。
そうした分け方をするだけでも、判断の重さはかなり変わります。

## 迷っている時間も、悪い時間ではない

退職金の使い方を1ヶ月考えても、長期投資の前提では大きな差にはなりにくいです。
焦って全額入れるより、自分が納得できるペースで始めるほうが続けやすいです。

退職金を投資に回す判断は、急ぐ必要がないことのほうが多いです。
迷っている日は、「まだ決めなくて大丈夫」と思えるだけで十分です。
