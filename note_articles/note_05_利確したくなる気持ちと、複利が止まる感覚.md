# 利確したいのに動けない。その「複利が止まる感覚」の正体

**利確したいのに決めきれず落ち着かない。売りたい気持ちと複利への不安が同時に来る理由を、この感覚の正体から解説します。**

## 利確したいのに動けない理由

資産が増えてくると、「どこかで確定させたい」という気持ちが出てきます。含み益が大きくなるほど、「戻ったらどうしよう」という不安も膨らみやすいものです。

頭では「長期で持つ」とわかっていても、指がアプリに向かうことがあります。それは誰にでもあることです。

利確してキャッシュになった瞬間、その分の複利の効果は一時的に途切れます。「また買い直せばいい」と思いやすいですが、売った後の再投資タイミングを正確に判断し続けることは難しいとされています。

JPモルガン・アセット・マネジメントの分析では、S&P500の過去20年間で上昇幅が大きかった上位10日を逃しただけで、年率リターンが約半分に低下する結果が示されています。

「逃げたい気持ち」と「離れたくない気持ち」が両方本物だからこそ、葛藤は生まれるのです。

## 売りたいときにすぐできること

売りたい気持ちが出てきたとき、すぐアプリを操作するのではなく、「積立設定が今も動いているか確認するだけにとどめる」ことをお勧めします。確認はしても、変更はしない。その一線を引くだけで、感情的な操作を止められることがあります。

証券口座を見る頻度を減らすことも、売買衝動を抑えやすくする方法として一般に言われています。

過去のデータでは、「売らずに持ち続けた期間が長いほど、複利の効果を受け取れた人が多い」とされています。
