# 教育費と投資の優先順位が決められないときの整理

**子どもの教育費も貯めたい、投資も続けたい。どちらを優先すべきか決められない。

その迷いを整理します。**。

## 迷うのは、どちらも「将来の自分のため」だから

教育費も投資も、目的は将来への備えです。
だから「どちらかを捨てる」と感じると、どちらも手放せなくなります。

この迷いは、お金の問題というより「どちらを後回しにしたら後悔するか」の不安です。
優先順位が決められないときは、金額の比較より「いつまでに必要か」で分けると整理しやすくなります。

## 期限がある方を先に確保する

教育費は使う時期がほぼ決まっています。
大学入学なら18歳、高校なら15歳。期限が明確です。

一方、投資の取り崩しは20年後、30年後かもしれません。
期限が近い教育費を先に確保して、残りの余力で投資を続ける。この順番だと迷いが減りやすいです。

## 投資額を減らしても、止めないほうが長期では効く

教育費を優先して投資額を月1万円に減らしても、仕組みは動き続けます。
止めるのと減らすのは、まったく違います。

教育費のピークが過ぎたあとに、投資額を戻すことはいつでもできます。
今すべてを同時に大きく進めようとするより、「時期で分ける」と考えたほうが、気持ちの余裕は残りやすいです。
