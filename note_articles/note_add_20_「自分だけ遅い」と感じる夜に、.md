# 「自分だけ遅い」と感じたら、比較を静かにする

**投資を始めたばかりの頃、他人の資産が気になることがあります。この記事では、その比較心理と向き合うための考え方をお伝えします。**

投資で他の人の成果が目に入りやすいのは、自然な心理です。「自分はまだこんなに少ない」という感覚を持つのは、長期投資をしている多くの人が経験しています。

このような比較があるからこそ、夜にモヤモヤして眠れなくなることもあります。

## 開始時期の差が、資産の差につながっている

同じ年齢の投資家でも、投資を開始した時期には大きなばらつきがあります。5年以上の差が珍しくないのです。

見えている現在の資産額の差は、「いつから始めたか」という開始時期の違いが影響しています。同い年だからといって、同じ期間投資をしているわけではないのです。

## 比較で苦しくなるのはなぜか

比較で苦しくなるのは、相手と自分が「同じタイミングで始まった」と無意識に思い込んでいるからです。

SNSに映る成果は、いつから始まったのかが見えていません。開始時期が異なる可能性に気づくだけで、落ち着きやすくなります。

## 自分の「期間」を記録する習慣

他の人の成果を見たときは、数字そのものより先に「自分は開始からどのくらい経っているか」を確認してください。

スマートフォンのメモ帳に「◯年◯月開始」と記録しておくと、そのたびに自分の経過期間を意識できます。「期間軸」を明確にすることで、比較の苦しさはかなり和らぎます。

長期投資で大事なのは、他の人との速さではなく、自分がどのくらいの期間続けてきたかです。
