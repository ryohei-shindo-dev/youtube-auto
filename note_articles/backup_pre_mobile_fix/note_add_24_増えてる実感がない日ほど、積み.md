# 増えてる実感がない日ほど、積み立ては効いている

増えてる実感がなくて積み立てに不安を感じる人へ。この記事では、残高が思ったほど伸びて見えない理由と、実感が薄い時期に積み立てが果たしている役割を整理します。

その感覚、実はとても一般的です。特に相場が動かない日や、ニュースで「市場横ばい」と報道されている時期は、自分の積み立てが本当に機能しているのか不安になりやすいものです。でも、その不安は実は、仕組みが正しく動いているサインかもしれません。

## 増えている実感がない時期に積み立てが効いている理由

金融庁のデータでは、長期積み立て投資の成果は「相場が安い時期の積み立ては、将来的な購入単価を下げる効果がある」とされています。つまり、相場が下がって「増えてない」と感じる時こそ、割安な価格でたくさん購入できているということです。逆に相場が好調な時期は、高い価格で少ない数量しか購入できていません。

つまり、増えてる実感がない日ほど、あなたの積み立ては確実に「安く仕込む」という仕事をしているのです。

## 増えていない実感が長期投資では自然な理由

心理的には、口座残高が目に見えて増えている時の方が気分が良いかもしれません。ですが、投資の観点では、むしろ反対です。相場が停滞している時期に「つまらないな」と感じながら積み立てを続ける人ほど、後年になって大きな恩恵を受ける傾向があります。

なぜなら、その「つまらない期間」に、あなたは何千株・何万口の安い資産を地道に買い集めているからです。

## 積み立ての実感がない日にできること

増えてる実感がない時は、むしろ口座を見る頻度を減らすと落ち着きやすいです。毎日チェックしていると「本当に大丈夫か？」という不安が増幅されます。週1回、あるいは月1回だけ確認するくらいに設定を変えると、相場の波に揺さぶられにくくなります。

また、毎月の積み立て設定をもう一度確認して「これは切らない」と心で決める。その小さな決断が、退場への誘惑を跳ね返す力になります。

相場が動かない日ほど、あなたの積み立ては最も効率的に機能しています。

## Q&A

**Q. 積み立てで「安く買える」というのは本当ですか？**
A. 相場が下がった時期には、同じ金額でより多くの口数が購入される仕組みです。これをドルコスト平均法と呼び、一般に長期積み立ての基本戦略とされています。
