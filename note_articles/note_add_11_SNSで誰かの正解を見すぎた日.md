# SNSで正解を見すぎた日に、心が疲れる理由

SNSで他人の投資成功を見ると、自分の選択が間違っているような気がしてきませんか。

## SNSで投資の正解を見すぎると心が疲れる理由

投資を始めると、SNSやYouTubeで「月利10%達成」「この銘柄で2倍になった」といった報告が目に入ります。それ自体は悪くないのですが、短時間に何十件も見ると、自分の長期投資計画が地味に感じられてくるんです。これは脳の仕組みによるもので、あなたの判断が悪いわけではありません。

## SNSでは投資の結果だけが見えやすい

心理学の研究では、SNSに投稿される情報は成功事例に大きく偏る傾向があるとされます。損失経験や失敗、含み損を抱えている時間は投稿されにくく、結果として成功ストーリーばかりが集中して表示される仕組みになっているのです。つまり、あなたが見ているのは全体像ではなく、フィルターがかかった一部の情報に過ぎません。

## SNSを見る頻度を減らすと投資の不安が落ち着きやすい

比較による疲れを減らすには、SNSを確認する回数や時間を決めておくと落ち着きやすくなります。例えば「週に1回だけ」「夜21時以降は見ない」など、自分でルールを作ってしまう方法です。投資情報は必要ですが、成功事例の動画まで毎日見る必要はありません。見る頻度を減らすことで、自分の積み立て計画が「地味ではなく、着実である」と改めて認識しやすくなります。

## 自分のペースを守ることが長期投資では大切です
