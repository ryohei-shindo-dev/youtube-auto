# 投資額を減らしたいけど、減らしたら負けな気がするときの整理

**積み立て額を減らしたい。でも減らしたら「逃げた」ことになる気がする。

その気持ちの整理です。**。

## 減らすことに罪悪感を感じるのは、続けることを正義にしすぎているから

「月5万を3万に減らしたい」と思ったとき、金額の問題よりも「減額=後退」という感覚が邪魔をします。

長期投資では「続けること」が大事だと言われます。
でもそれが「金額を維持すること」にすり替わると、生活を圧迫してでも続けなければという重さになります。

## 減額しても、止めるよりずっと強い

月5万を月1万に減らしても、積み立てが続いていれば仕組みは動いています。
止めることと減らすことは、まったく別のことです。

生活費を削って投資を維持するより、金額を落としてでも続けるほうが、長期では崩れにくいです。
「続ける」の定義を「同じ金額を維持する」から「仕組みを止めない」に置き換えると、気持ちが少し軽くなります。

## 減らしたくなったとき、確認するのは1つだけ

「減らしたい」と感じたとき、確認するのは1つだけで十分です。
「生活に余裕がないから減らすのか、不安だから減らすのか」です。

生活の問題が大きいなら、減額は自然な調整と考えられます。
不安の問題が大きいなら、金額より口座を見る頻度を変えるほうが効くこともあります。

どちらにしても、減らすことをそのまま負けと結びつけなくて大丈夫です。
