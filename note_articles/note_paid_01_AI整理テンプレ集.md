# 投資で揺れた日に使える、AI整理テンプレ集｜ChatGPTに投資相談する前の質問例

含み損が気になる夜。比較疲れで商品を変えたくなる日。積み立てをやめようか迷う週末。

そんなとき、ChatGPTやCopilotに「どうすればいいですか？」と聞いたことはありませんか。

でも、返ってきた答えを読んでも、なぜかモヤモヤが消えない。むしろ余計に迷う。そんな経験がある方に向けて、この記事を書きました。

## 1. AIに投資相談すると、なぜ話がずれやすいのか

AIに投資の悩みを相談するとき、多くの人が「何を買えばいいですか」「売ったほうがいいですか」と聞きます。

AIはそれに対して、一般論で答えます。「分散投資が大切です」「長期的に見れば回復する傾向があります」「リスク許容度を確認しましょう」。

間違ってはいません。でも、あなたが本当に聞きたかったことには届いていない。

なぜか。

あなたが抱えているのは「知識の不足」ではなく、「気持ちの揺れ」だからです。

含み損が気になるのは、損失額を知らないからではありません。数字は見えている。それでも気になる。その「気になる」の正体を言葉にしないまま質問すると、AIは知識で答えるしかなくなります。

## 2. 「答えをください」だと苦しくなりやすい理由

AIに「正解を教えてください」と聞くと、AIは正解っぽいものを返します。

でも投資の悩みには、多くの場合「正解」がありません。あるのは「自分がどこまで納得できるか」の線引きだけです。

だから、AIの使い方を変える必要があります。

**「答えをもらう」のではなく、「整理を手伝ってもらう」。**

「自分は今、何に不安を感じているのか」
「その不安は、数字の問題なのか、気持ちの問題なのか」
「どこまでなら受け入れられるのか」

この整理を、AIに手伝ってもらう。そのために必要なのが、聞き方のテンプレートです。

## 3. このテンプレ集の使い方

この記事には、投資で気持ちが揺れやすい場面ごとに、AIへの質問テンプレートをまとめています。

使い方はシンプルです。

1. 自分の気持ちに近い場面を選ぶ
2. テンプレートをコピーしてChatGPT等に貼り付ける
3. 【】の中を自分の状況に書き換える
4. AIの回答を読んで、自分の気持ちを整理する

大事なのは、AIの回答を「正解」として受け取らないことです。AIが返すのは整理の補助線であって、最終的に決めるのは自分自身です。

また、このテンプレートは**投資判断そのものを提供するものではありません**。個別の銘柄選びや売買タイミングの判断には使えません。あくまで「自分の気持ちを言葉にして、整理するための道具」です。

## 4. このテンプレ集で扱う悩み

以下の8つの場面に対応したテンプレートを用意しました。

- 含み損が気になって眠れない夜
- 比較疲れで商品を乗り換えたくなるとき
- 積み立てをやめたくなったとき
- 情報が多すぎて疲れたとき
- 老後資金が間に合うか不安なとき
- 新NISAで何を買うか迷っているとき
- 損切りしたくなったとき
- 全部売ったあと、戻れなくなっているとき

どれも、このnoteの無料記事で扱ってきたテーマです。
無料記事では「考え方」を整理してきました。
ここから先では、それを自分の状況に当てはめるための具体的な質問文として、すぐ使える形にまとめています。
「読むだけ」では整理しきれない日に使うためのパートです。

---

※本記事は、投資に関する一般的な整理の補助を目的としたものであり、特定の金融商品、売買タイミング、投資判断を推奨するものではありません。

ここから先は有料パートです。8つの場面ごとに、そのまま使える質問テンプレートと、AIの回答を読むときの注意点をまとめています。

## 5. 含み損が気になる夜の整理テンプレ

**こんなとき:** 評価額がマイナスになっていて、画面を開くたびに気になる。売ったほうがいいのか、持ち続けていいのか分からない。

**テンプレート:**

```
私は【投資信託/ETF等の種類】に積み立て投資をしています。
現在、評価額が【○万円/○%】のマイナスになっています。
投資を始めてから【○年/○ヶ月】です。

今の気持ちとしては、【毎日口座を確認してしまう/売ったほうがいいか迷っている/このまま続けていいのか不安】です。

以下の3つの視点から、私の状況を整理してください。
1. この含み損は、積み立て投資の過程でよくある範囲かどうか
2. 私が不安に感じている原因は、金額の大きさなのか、それとも別の要因がありそうか
3. 今の状況で「何もしない」という選択肢は、どういう意味を持つか

※投資判断のアドバイスは不要です。気持ちの整理を手伝ってください。
```

**AIの回答を読むときの注意:**
- AIが「一般的には長期で回復する傾向があります」と言っても、それはあなたの状況の保証ではありません
- 「不安の原因は金額ではなく確認頻度かもしれません」という指摘が出たら、そこを深掘りしてみてください
- 「何もしない」の意味を自分なりに言葉にできたら、それだけで十分です

## 6. 比較疲れで商品を乗り換えたくなるときの整理テンプレ

このテンプレートは、すでに持っている商品を変えたくなっているとき向けです。

**こんなとき:** S&P500とオルカンで迷っている。他の人の運用成績を見て、自分の選択が間違っている気がする。

**テンプレート:**

```
私は現在【オルカン/S&P500/その他】に積み立てています。
最近、【S&P500/オルカン/NASDAQ100/高配当/その他】のほうが成績がいいと知り、乗り換えを検討しています。

乗り換えたい理由は【直近のリターンが高い/周りが買っている/自分の商品が増えていない気がする】です。

以下を整理してください。
1. 私が比較しているのは、同じ期間・同じ条件での比較か
2. 乗り換えたい気持ちの裏にある本当の不安は何か
3. 「今の商品を選んだ最初の理由」を思い出すことに意味があるか

※商品の優劣を判定してほしいわけではありません。自分の判断の軸を整理したいです。
```

**注意:**
- AIが特定の商品を推奨するような回答をした場合、それは参考にしないでください
- 「最初に選んだ理由」を自分で書き出すだけでも、かなり整理が進みます

## 7. 積み立てをやめたくなったときの整理テンプレ

**こんなとき:** 毎月の引き落としが重く感じる。成果が見えない。やめたい気持ちと、やめたら後悔する気持ちの間で揺れている。

**テンプレート:**

```
積み立て投資を始めて【○年/○ヶ月】です。
毎月【○万円】を積み立てていますが、最近やめたいと感じています。

やめたい理由は【成果が見えない/生活費が苦しい/他にお金を使いたい/意味があるか分からなくなった】です。

以下の視点で整理してください。
1. 「やめたい」と感じているのは、金額の問題か、気持ちの問題か
2. 金額を減らす（やめるのではなく）という選択肢はどういう意味を持つか
3. 3ヶ月前の自分は、積み立てについてどう感じていたか振り返る意味はあるか

※続けるべきか、やめるべきかの判断は求めていません。
```

**注意:**
- 生活費が苦しいなら、金額を見直すこと自体は自然な検討対象です。それは「負け」ではありません
- AIの回答が「続けたほうがいいです」と断言していたら、その部分は無視してください

## 8. 情報が多すぎて疲れたときの整理テンプレ

**こんなとき:** YouTube、SNS、ニュース、本。情報を集めれば集めるほど、何が正しいか分からなくなった。

**テンプレート:**

```
投資の情報収集をしていますが、情報が多すぎて疲れています。
主に【YouTube/X(Twitter)/ニュースサイト/本/SNS】から情報を得ています。

今感じていることは【何が正しいか分からない/人によって言うことが違う/見るたびに不安になる/情報を見ないと不安になる】です。

以下を整理してください。
1. 私が本当に必要としている情報は何か（知識なのか、安心なのか）
2. 情報収集を「減らす」ことのメリットとデメリット
3. 「週に1回だけ確認する」ルールを作った場合、何が変わりそうか

※おすすめの情報源を教えてほしいわけではありません。
```

**注意:**
- 「情報を見ないと不安」は、情報不足ではなく確認癖の可能性があります
- AIが情報源のランキングを出した場合、それは無視して構いません

## 9. 老後資金が間に合うか不安なときの整理テンプレ

**こんなとき:** 65歳までに必要な金額と、今のペースの差が大きすぎて不安。

**テンプレート:**

```
現在【○歳】で、【○歳】までに老後資金を準備したいと考えています。
目標額は【○万円】で、現在の積み立てペースは月【○万円】です。
現在の資産は【○万円】程度です。

不安に感じていることは【間に合わない気がする/計算するほど不安になる/他の人と比べて遅い気がする】です。

以下を整理してください。
1. 「間に合わない」という感覚は、計算上の事実に基づいているか、漠然とした不安か
2. 目標額の根拠は何か（メディアの数字をそのまま使っていないか）
3. 「完璧に間に合わなくても、部分的に準備できていること」の意味

※具体的な運用利回りの試算は不要です。気持ちの整理を優先してください。
```

**注意:**
- 老後2000万円問題の数字は、すべての人に当てはまるわけではありません
- AIが具体的な利回り計算をした場合、その数字は前提次第で大きく変わります

## 10. 新NISAで何を買うか迷っているときの整理テンプレ

このテンプレートは、まだ始める前に商品選びで止まっているとき向けです。

**こんなとき:** つみたて投資枠と成長投資枠の使い分けが分からない。商品が多すぎて決められない。

**テンプレート:**

```
新NISAを始めようとしていますが、何を買うか決められません。
迷っている選択肢は【オルカン/S&P500/バランス型/個別株/高配当ETF/その他】です。

迷っている理由は【どれが一番得か分からない/失敗したくない/人によって言うことが違う/決めたあとに後悔しそう】です。

以下を整理してください。
1. 私が本当に迷っているのは「商品選び」なのか、「失敗への恐れ」なのか
2. 「完璧な商品」を探し続けることのコスト（時間と気力）
3. 「まず1つだけ選んで始める」ことの意味

※具体的な商品を推薦しないでください。判断の軸を整理したいだけです。
```

**注意:**
- AIが特定の商品を推奨した場合、それは無視してください
- 「決められない」こと自体が悪いわけではありません。でも、決められないまま時間が過ぎることにはコストがあります

## 11. 損切りしたくなったときの整理テンプレ

**こんなとき:** 含み損が大きくなって、これ以上の損失を避けたい。いったん売って仕切り直したい。

**テンプレート:**

```
現在、【投資信託/ETF/個別株】で【○万円/○%】の含み損があります。
投資を始めてから【○年/○ヶ月】です。

今の気持ちは【これ以上の損失が怖い/いったん全部売ってリセットしたい/ニュースを見るたびに不安が増す】です。

以下を整理してください。
1. 「今売りたい」という気持ちは、合理的な判断から来ているのか、感情的な反応なのか
2. もし今売った場合、そのあと何をするつもりか（再投資するのか、現金で持つのか）
3. 「売らない」ことと「何もしない」ことは同じか、違うか

※売るべきか、持つべきかの結論は不要です。
```

**注意:**
- 「売ったあとどうするか」を言葉にしづらいときは、判断を急いでいる可能性があります
- AIが「長期投資なら持ち続けるべき」と言っても、それはあなたの状況を知った上での判断ではありません

## 12. 全部売ったあと、戻れなくなっているときの整理テンプレ

**こんなとき:** 怖くなって全部売った。相場は戻った。でも再開する勇気が出ない。

**テンプレート:**

```
以前、【投資信託/ETF等】に投資していましたが、【暴落/含み損/不安】で全部売却しました。
売却してから【○ヶ月/○年】が経っています。
その後、相場は【回復した/横ばい/さらに下がった】状況です。

今の気持ちは【再開したいけど怖い/売ったことを後悔している/もう投資に向いていないと思う】です。

以下を整理してください。
1. 「全部売った」ことへの後悔と、「再開する恐怖」は別の感情かどうか
2. 再開するとしたら、前回と同じやり方でいいのか、変えるべきことがあるか
3. 「小さく再開する」という選択肢はどういう意味を持つか

※再開すべきかどうかの判断は求めていません。気持ちの整理だけお願いします。
```

**注意:**
- 全部売った判断を責める必要はありません。そのときの自分にとっては、それが精一杯の判断でした
- 「小さく再開する」という考え方は、負担を抑えながら感覚を戻す整理として使えます

## 13. AIの回答を読むときに見落としやすいこと

テンプレートを使ってAIに聞いたあと、回答を読むときに覚えておいてほしいことがあります。

**AIは「あなたの状況」を知りません。**
テンプレートで状況を伝えても、AIが知っているのはその文面だけです。あなたの家族構成、収入、性格、過去の経験は分かりません。だから、AIの回答はあくまで「一般的な整理の補助線」です。

**AIは「大丈夫です」と言いがちです。**
安心させようとする傾向があります。「大丈夫です、長期的には回復する傾向があります」という回答は、あなたへの保証ではありません。

**AIの回答で一番使えるのは「質問」です。**
AIが返す「答え」よりも、AIが投げかけてくる「あなたはどう思いますか？」「なぜそう感じますか？」という問いかけのほうが、整理に役立ちます。

**1回で終わらなくていい。**
テンプレートを使って1回聞いて、AIの回答を読んで、「ここはちょっと違う」と思ったら、そこをさらに聞き直してください。対話を重ねるほど、自分の考えが見えてきます。

| 含み損が気になる夜 | 「私は【投資信託/ETF等の種類】に積み立て投資をしています。現在、評価額が…」 |
| 比較疲れで乗り換えたい | 「私は現在【オルカン/S&P500/その他】に積み立てています。最近…」 |
| 積み立てをやめたい | 「積み立て投資を始めて【○年/○ヶ月】です。毎月【○万円】を…」 |
| 情報が多すぎて疲れた | 「投資の情報収集をしていますが、情報が多すぎて疲れています…」 |
| 老後資金が間に合うか | 「現在【○歳】で、【○歳】までに老後資金を準備したいと…」 |
| 何を買うか迷っている | 「新NISAを始めようとしていますが、何を買うか決められません…」 |
| 損切りしたい | 「現在、【投資信託/ETF/個別株】で【○万円/○%】の含み損が…」 |
| 全部売ったあと戻れない | 「以前、【投資信託/ETF等】に投資していましたが…」 |

すべてのテンプレートに共通するのは、最後の一文です。

**「※投資判断のアドバイスは不要です。気持ちの整理を手伝ってください。」**

この一文を入れるだけで、AIの回答の質が変わります。

まずは、いちばん頻繁に揺れる場面のテンプレートを1つだけ保存してください。
すべてを使いこなす必要はありません。
「この場面では、この聞き方に戻る」という1本があるだけでも、揺れた日の負担はかなり変わります。
