# 下がっているときに追加投資すべきか迷うときの整理

**相場が下がっている。追加で買えばチャンスかもしれない。

でも、もっと下がるかもしれない。その迷いを整理します。

**。

## 「安く買えるチャンス」と「もっと下がる恐怖」が同時に来る

下落局面では、2つの声が頭の中で同時に鳴ります。
「今が安い。買い増すべきだ」と「まだ底じゃない。もっと下がる」です。

どちらも正しいように聞こえるから、判断ができない。
この迷いは、知識が足りないから起きるのではなく、未来が分からないから起きます。

## 追加投資は、「正解の底」を当てる話ではなく「自分が崩れない金額」の話

追加投資を「今が底か」で考えると、答えは出にくくなります。
底がいつかは、あとにならないと分からないからです。

代わりに、「この金額なら下がっても持ち続けられるか」で考えると、判断の基準が少し変わります。
勝てるかどうかより、崩れないかどうかで見るほうが、この迷いには合いやすいです。

## 迷うなら、積み立て設定をそのまま続けるだけでも十分

追加投資をしなくても、毎月の積み立てが動いていれば、下がった月には自動で安く買えています。
ドルコスト平均法はそもそも、追加判断をしなくて済む仕組みです。

「下がっているから何かしなきゃ」と感じるときほど、何もしないことが仕組みを活かすことになります。
迷っている日は、設定をそのままにして、判断を増やさないだけで十分です。
