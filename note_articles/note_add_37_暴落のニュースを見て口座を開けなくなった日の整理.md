# 暴落のニュースを見て口座を開けなくなった日の整理

**暴落のニュースが流れると、口座を見るのが怖くなる。見たくないのに気になる。

その状態を整理します。**
## 口座を開けないのは、数字を見たくないのではなく「現実にしたくない」から

暴落のニュースを見たあと、口座を開けなくなることがあります。
これは臆病なのではなく、「数字を見た瞬間に損が確定する気がする」からです。

実際には見ても見なくても含み損は変わりません。
でも、見ることで「現実にしてしまう」感覚が怖い。この気持ちは自然な反応です。

## 見ない日があっても、積み立ては動いている

口座を開けなかった日でも、積み立ての設定は止まっていません。
自動で引き落とされ、自動で買い付けが行われています。

見なかったことで何かを失ったわけではありません。
暴落の日は、数字を見てすぐに判断したくなることがあります。
そう考えると、見ないで一日やり過ごすことが、気持ちを守る時間になることもあります。

## 開けるようになったときに確認すること

気持ちが落ち着いてから口座を開いたとき、確認するのは1つだけで十分です。
「積み立ての設定が動いているか」です。

評価額がいくら下がったかは、長期では一時的な数字です。
設定が止まっていないことが確認できれば、その日はそれで十分です。
