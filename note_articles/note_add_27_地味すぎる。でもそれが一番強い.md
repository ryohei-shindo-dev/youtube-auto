# 派手さゼロ。でも20年後、それが最強になる

派手な投資法は目立つけれど、地味な積み立てが一番強いって、なぜか信じられなくなる時があります。

SNSで「○○で◎倍になった」という話を見ていると、自分の毎月の積み立てが本当にちっぽけに思えてしまいます。コツコツしているのに、周りは大きく動いている気がして、焦りを感じるのは自然なことです。でも、その焦りは実は、データが教えてくれる現実とは異なる可能性があります。

## 派手な投資方法ほど続きにくい理由

米国の投資行動研究では、年間売買回数が多い投資家ほど、実際のリターンが低くなる傾向が報告されています。つまり、頻繁に取引する（派手な投資をする）ほど、手数料や判断ミスで成績が落ちやすいということです。一方、積み立てを淡々と続ける方法は、取引回数が少ないぶん余計な損失が減る仕組みになっています。

なぜなら、派手な投資法は「勝つ時」は目立ちますが、「負ける時」も激しいからです。その上下動を何度も経験すると、人間の心は疲弊して、ルール通りに続けることが難しくなります。地味な方法は上下動が穏やかなので、心理的に続けやすい設計になっているのです。

## 20年後に差が出やすいのは毎月変わらず続けることです

もし今、周囲の派手な話に揺らいでいるなら、試してほしいのは「投資の結果を見る頻度を決める」ことです。毎日見ていれば、小さな上下動が大きく見えます。週1回、あるいは月1回と決めておくと、短期的なノイズではなく、中長期的なトレンドだけが目に入るようになります。

また、積み立てのルール（毎月◎日に◎万円など）を一度決めたら、その後は「淡々と実行する」以外のアクションを取らないことです。決め直す、やめる、増やす……そうした判断を何度も重ねることが、実は心も体力も消耗させます。一度決めたら、変わらずを繰り返すことが、実は最も強い投資姿勢だと整理できます。

## まとめ

派手さがない分、誰の目にも付きませんが、過去のデータでは、その方法が最も成績につながる傾向があります。
