# オルカンを持っているのに、置いていかれる気がするとき

オルカン一本で大丈夫なはずなのに、SNSで「あの銘柄が上がった」という投稿を見ると、急に不安になってしまう。

長期投資を始めた人なら、このモヤモヤを経験したことがあるのではないでしょうか。オルカン（全世界株式インデックスファンド）を淡々と積み立てているのに、世界中の誰かが別の選択肢で大きな利益を出している。そのニュースが目に入るたびに「自分は本当に正しい選択をしているのか」という疑問が湧く。その気持ちはとても自然なものです。

## オルカンを持っているのに置いていかれる気がする理由

過去20年間の先進国株式インデックスと個別株パフォーマンスの研究によれば、5年単位で見ると個別株が上回る年も多くありますが、20年単位で見ると約90%の個別株投資家がインデックスに負けたという統計があります。SNSで目立つのは、その短期の上昇局面に当たった運と判断が重なった人たちであり、多くの人の長期成績は見えていないということです。

つまり「置いていかれている」と感じるのは、短期的な勝者の目立つ情報だけを見ているためかもしれません。全体像で考えると、オルカンのような分散投資は統計的に安定した成果を生む傾向があります。

## オルカンでいいのか迷ったときに確認したいこと

不安になったときは、自分が選んだオルカンを改めて見つめ直す時間を作ると良いでしょう。具体的には、自分の家計と運用期間を紙に書き出し、「月いくら積み立てられるのか」「何年続ける予定か」を整理するだけでも、置いていかれているのではなく「自分のペースで正しく進んでいる」という実感が戻りやすくなります。

焦りの正体は、比較による不安です。その不安を静めるには、他人の短期の成果ではなく、自分の長期計画に目を向けることが効果的です。

オルカン一本だからこそ、迷わず続けられる。その単純さが、実は最大の強みです。
