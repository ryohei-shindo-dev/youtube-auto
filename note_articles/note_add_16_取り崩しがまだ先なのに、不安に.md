# 取り崩しまで20年あるのに、夜眠れなくなるのはなぜ

市場が下がった日の夜、資産状況を見ると心が沈んで、寝付けなくなってしまう。

長期投資を始めて数年。取り崩しまではまだ20年以上あるはずなのに、ここ最近の相場変動で不安が増してきた。ネット上の悲観的な声を見ると「今のうちに売却すべきでは」と考え始め、夜中に家計管理アプリを開いてしまう。そんな経験はありませんか。取り崩しまでに十分な時間があるはずなのに、なぜか心が落ち着かない。その感覚は決して珍しくありません。

## 「今」の変動が、未来の全てに見える

私たちの脳は、直近の出来事に強く反応する傾向があります。行動経済学の研究では、投資家は過去1年間のリターンを強く意識する傾向があることが報告されています。つまり、数ヶ月の下落を見ると、それが今後ずっと続くように感じてしまうということです。この心理的な錯覚は、時間軸の長い長期投資者ほど、実は陥りやすいものです。

なぜなら、長期投資は頭では「20年は大丈夫」と理解していても、目の前の数字は毎日変わっているから。毎日変わる「今」と、20年後という「いつか来る日」のギャップが、不安を増幅させるのです。

## 定期的な確認を、「定期的」に保つ

夜眠れなくなるほど不安な時は、ポートフォリオを確認する頻度を意識的に減らすと落ち着きやすいとされています。毎日確認する習慣がついていれば、それを週1回、月1回に変えてみる。あるいは、市場が大きく動いた日の夜間帯だけ見ないルールを作るのも効果的です。

同時に重要なのは、積み立ての流れだけは止めないようにすることです。毎月の積み立てを続けることで、脳が「私は計画通り進めている」という安心感を得やすくなります。相場と関係なく続く定期的なアクションが、逆説的に心の定期的な安定をもたらすのです。

## 夜の不安は、設定した計画が正しいサイン

取り崩しまで20年ある状況での夜間の不安は、むしろあなたが長期投資のルールを守ろうとしている証です。短期で売買する人は、こうした夜間の不安を感じる暇もありません。定期的に現在地を確認しながら、その時間を少し減らしていく工夫が、心の余裕をもたらします。

## よくある質問

**Q1. 毎日見てしまう癖を治すコツはありますか？**
アプリを削除するのではなく、スマホのホーム画面から見えない場所に移すと効果的です。決められた曜日だけ確認するといった「仕儀式化」も、心理的な落ち着きをもたらしやすいとされています。

**Q2. 夜間に不安で起きてしまったら、どうすればいいですか？**
その時点で資産状況を確認するのは避けた方が落ち着きやすいです。計画書や積み立て設定の確認など、未来のアクションを再度目で追う方が、心の整理につながる傾向があります。

**Q3. 取り崩しまでの期間が長いほど、こうした不安は続きますか？**
期間が長いほど、相場変動の中で複数回の不安が訪れる可能性があります。重要なのは「不安が来ること」が正常であり、その度に立ち止まるのではなく、定期確認の頻度を調整することです。
