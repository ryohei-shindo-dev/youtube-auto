# 品質レビュースキル

生成済み動画・記事の品質をチェックするワークフロー。

## Shorts レビュー

1. `review_gen.py` を実行してレビュー用プロンプトを生成
   ```bash
   python review_gen.py
   ```
2. クリップボードの内容を ChatGPT に貼り付け
3. スコア（10点満点）を Google Sheets の O-U 列に記録

## チェック項目

### コンテンツ（AGENTS.md 準拠）
- [ ] hook が感情を止める痛みワードか（説明ではなく感情）
- [ ] 感情曲線: 痛み → 共感 → データ → 安心 の順か
- [ ] 投資助言と受け取られる表現がないか
- [ ] 煽り・誇張・過剰断定がないか（「絶対儲かる」等）
- [ ] 尺が 12〜15 秒に収まっているか

### 技術
- [ ] 音声の読み間違いがないか
- [ ] スライドの文字が切れていないか（禁則処理）
- [ ] BGM と音声のバランスが適切か

## note 記事レビュー
- 800〜1500 字に収まっているか
- 静かで断定しすぎない語り口か（note_articles/*.md を参考）
- 免責表記（投資助言ではない旨）が含まれているか
