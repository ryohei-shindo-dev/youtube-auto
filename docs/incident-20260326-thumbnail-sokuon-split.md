# サムネフレームで促音（っ）の後ろで改行

## 症状

3/26 投稿の YouTube Shorts「テンバガー狙って疲れてませんか？」のサムネフレームで、テキストが「テンバガー狙っ / て疲れてません」と促音（っ）の後ろで改行されていた。

## 根本原因

`slide_gen.py` の `_NO_TAIL_CHARS`（行末禁止文字）に促音「っ」「ッ」が含まれていなかった。

`_find_split_pos()` が分割位置を探す際、「狙っ」の「っ」の後ろが有効な分割位置と判定され、「狙っ / て疲れてません」という不自然な改行が生成された。

## 修正内容

`_NO_TAIL_CHARS` に `っ` と `ッ` を追加。

```python
# 修正前
_NO_TAIL_CHARS = set("（「『【〈《([{第約")

# 修正後
_NO_TAIL_CHARS = set("（「『【〈《([{第約っッ")
```

修正後の分割結果: 「テンバガー狙って / 疲れてませんか？」（助詞「て」の後で分割）

## 影響範囲

- 3/26 投稿分のサムネフレーム1本のみ（既に公開済み）
- キュー内の残り62本には同様の問題なし（確認済み）
