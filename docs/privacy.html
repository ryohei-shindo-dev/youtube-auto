Privacy Policy

Last updated: 2026-03-06

This application ("gachiho-motive") accesses TikTok user data solely for the purpose of publishing short-form investment motivation videos on behalf of the authenticated user.

Data Collection

We collect only the minimum data required for authentication (TikTok user ID and access tokens). No personal data is shared with third parties.

Data Usage

Access tokens are stored locally and used exclusively to upload videos to the authenticated user's TikTok account.

Data Deletion

Users may revoke access at any time through TikTok's settings. Upon revocation, all stored tokens are automatically invalidated.

For questions, please contact us via our YouTube channel.
