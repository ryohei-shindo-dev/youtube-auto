# ChatGPT相談: Xアカウントに「偽装行為」ラベルが付いた

## 何が起きたか

Xアカウント @gachiho_motive に「偽装行為」ラベルが付けられた。
警告内容:
- 「アカウントがスパムを含んでいるか、その他の種類の偽装行為に関与している可能性がある」
- 影響: トレンドや検索結果から除外、表示範囲が限定、コンテンツが一時的に制限される可能性

## アカウントの状況

- チャンネル名: ガチホのモチベ（長期投資の継続モチベーション）
- 初投稿: 2026/03/06
- 約8日間で20本のX投稿済み
- 1日2回（07:30と19:30）にAPI経由で自動投稿
- 別途、1日1回（21:30）にnote記事のX告知を自動投稿
- つまり1日最大3回のAPI投稿

## 自動投稿の仕組み

### Shorts告知（1日2回）
- launchdで07:30と19:30に `auto_publish.py` を実行
- YouTube投稿後にX API v2でテキスト+YouTube URLを投稿
- 投稿内容例:
  ```
  含み損。

  でも20年続けた人
  元本割れゼロ。

  #長期投資 #ガチホ

  今日のShorts👇
  https://youtube.com/shorts/xxxx
  ```

### note記事告知（1日1回）
- launchdで21:30に `note_x_announce.py --x-only` を実行
- 公開済みnote記事のURLをXで告知
- 投稿内容例:
  ```
  📝 note更新しました

  「積立3年目が一番つらい理由」

  https://note.com/gachiho_motive/n/xxxx

  #長期投資 #ガチホ
  ```

### X API認証
- OAuth 2.0 PKCE（User Context）
- X API v2 (`POST /2/tweets`)
- Free Tierの可能性あり（API利用プランは未確認）

## 考えられる原因

1. **短期間での大量API投稿**: 8日間で約20本＋note告知10本前後 = 計30本程度
2. **機械的な投稿パターン**: 毎日同じ時刻に同じフォーマットで投稿
3. **テンプレ投稿の類似性**: ハッシュタグ・フォーマットが毎回ほぼ同じ
4. **アカウントが新しい**: 2026/03/06作成で、いきなりAPI自動投稿を開始
5. **Free Tier API の制限**: Free Tierは月1,500ツイートまでだが、自動投稿の審査が厳しい可能性
6. **人間の操作がゼロ**: 一度も手動ツイートしていない（API経由のみ）
