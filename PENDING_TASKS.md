## 1. TikTok 本番審査の通過対応

**状態**: TikTok 2回目も否認（2026-03-16）。API自動投稿はいったん断念。

**否認理由**: 個人利用・社内利用不可。Content Posting API は第三者ユーザー向けプラットフォーム用であり、自社アカウントへの自動投稿は想定用途外（ChatGPT評価 2026-03-16）。

**方針**: TikTok API 再申請は止める。手動投稿前提で最低限アカウントを育てる。YouTube / Instagram を優先。
