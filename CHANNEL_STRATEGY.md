## Shorts台本構成（5シーン・12〜15秒）

感情曲線: **痛みワードで掴む → 共感 → データで安心 → 断言・希望 → 余韻で終わる**

| # | ロール | 秒数 | 目的 | slide_text |
|---|---|---|---|---|
| 1 | hook | 2秒 | 痛みワード1語で掴む | 6文字以内（暴落。含み損。売りたい。） |
| 2 | empathy | 3秒 | 共感（語りかけは15%のみ） | ナレーション要点 |
| 3 | data | 4秒 | 具体的数字で安心材料 | 22文字以内 |
| 4 | resolve | 3秒 | 断言・希望 | 5〜7文字（時間が武器。退場＝負け。） |
| 5 | closing | 2秒 | 余韻（CTAは25%のみ、弱めの表現） | resolveの余韻 or 弱CTA |

### 最強台本テンプレ（ChatGPT分析）

「7秒で意味が完結する構造」が最も再生が伸びる。言葉の長さを固定すると安定。

```
hook（1秒）: 1〜2語     例: 含み損 / 売りたい / 暴落
empathy（2秒）: 共感    例: あなただけじゃない / まだ残ってますか
data（4秒）: 数字       例: 20年続けた人、元本割れゼロ
resolve（2秒）: 結論    例: 時間が最大の武器 / 退場したら終わり
closing（2秒）: 質問    例: 続けてますか / 同じ人いますか
```

完成例（約12秒）:
```
含み損。あなただけじゃない、まだ残ってますか。
20年続けた人、元本割れゼロ。時間が最大の武器。続けてますか。
```

### hookの強さランキング
強い: ① 含み損 ② 暴落 ③ 売りたい ④ 不安 ⑤ 退場
弱い: 複利 / 分散投資 / 資産形成（感情が弱い）

### closing方針（2026-03-16改定）
- 基本はCTAなし。resolveの余韻で終える
- 4本に1本だけ弱いCTA: 「また必要なときにどうぞ」「同じように続けている人へ」等
- 強いCTA（「フォローお願いします」「コメントで教えてください」）は使わない
- ブランドは「入口」ではなく「視聴後感」で統一する

## 投稿順ルール

1. **hookが強いものを優先**（含み損 > 暴落 > 売りたい > 不安 > 焦り）
2. **同ジャンル連続を避ける**（ラウンドロビンでhook分散済み）
3. 理想の並び: 暴落→含み損→積立→売りたい→時間→...

## アルゴリズム攻略5つ

1. **最初の1秒** — 痛みワード1語で視聴継続率を上げる
2. **12〜15秒** — 完走率が高い＝アルゴリズム評価UP
3. **ループ再生** — closing→hookが自然に繋がると視聴回数増（今後対応）
4. **1日2本投稿** — Shorts最適頻度
5. **同ジャンル連続を避ける** — テーマ分散で評価維持

## 映像・演出方針

### スライド画像（v2写真型）
- ストック写真（5カテゴリ×約60枚 = 274枚）を使用
- 縦型写真: 全画面写真 + 下部グラデーション + テキスト重ね（没入型）
- 横型写真: 上部55%写真 + 下部45%テキスト（分割型）
- 色補正はロール別（hook=やや暗(0.94) → closing=ほぼ原色(1.02)、感情曲線に連動）
- テキスト折り返し: 数字+単位の分断防止、禁則処理あり

### 字幕
- Pillowでスライド画像に焼き込み（下から420px位置、YouTubeUI回避）
- 白文字 + 黒縁（ストローク3px）
- 15文字/行で改行

### AI音声
- ElevenLabs eleven_v3（2026-03-16切替、全84本再生成済み）
- speed=1.15（通常）/ 1.05（closing）
- stability=0.4（通常）/ 0.65（closing）
- 句点後に読点「。、」でポーズ強制（v3対策）
- 投資用語の読み替え辞書あり（390+エントリ）

### BGM
- assets/bgm_ambient.m4a（静かなアンビエント）
- 音量: BGM_VOLUME=0.08（ナレーションの邪魔をしない）
- ループ再生、フェードイン1秒

### resolve前の間
- 0.7秒の無音クリップ挿入（断言前の「間」演出）

## コンテンツポリシー

### 絶対にやらないこと
- 個別株の推奨

## 収益化戦略（2026-03-20 ChatGPT分析に基づき更新）

### 基本方針
- **広告だけで回収する型ではなく、信頼を積み上げて複数の収益源を重ねる型**
- Shorts広告は構造的に薄い（Shorts Feed Ads、クリエイター取り分45%、RPMも低い）
- 三層設計: **Shorts = 新規流入** / **長尺 = 審査補強+信頼形成** / **note = 深掘り+課金**
- 長尺は「広告の柱」ではなく「審査補強と信頼形成の証拠」として運用する

### YPP到達の現実（2026-03-20時点の試算）
- 拡張YPP: 500登録 + Shorts 300万回/90日
- フルYPP: 1,000登録 + Shorts 1,000万回/90日
- 現ペース: 35本で約4万再生（14日） → 90日換算で約25.7万再生
- 300万回には現在の約11.7倍、1,000万回には約38.9倍が必要
- **今のボトルネックは収益設計より到達率の伸び。まず再生効率を上げるしかない**

### inauthentic content リスク（重要）
- YouTube は 2025年7月に「repetitious content」を「inauthentic content」と明確化
- 反復的・量産的に見える作りは収益化審査で不利になりえる
- AI音声・自動化自体が即NGではないが、独自性と一貫した人格が重要
- 対策1: テーマ重複回避（dedupe_check.py）
- 対策2: **フォーマットの見た目差分を増やす**（1枚絵回・データ強調回・比較図回・静かな語り回・長尺切り出し回など、外見上も別シリーズに寄せる）
- 対策3: 人間の編集工程の証拠を残す（A/Bテストログ・タイトル差し替え履歴・サムネ改善履歴・台本修正履歴）
- 対策4: 長尺を月1本でも続ける（「短尺テンプレの量産だけではない」という補強）

| 500〜1,000人 | **note単品有料を小さく試す**（低単価300〜500円）。拡張YPP申請 | note単品有料（少額） |
| 〜5,000人 | 単品有料の反応を見てメンバーシップ開始を判断。YouTube Shopping affiliate | 長尺広告 + note有料 + メンバーシップ |
| 〜1万人 | メンバーシップ本格化。コミュニティ型月額課金 | 上記の拡大 |
| 〜10万人 | 自社商品・提携企画・書籍 | 広告 + note + メンバーシップ + 商品 |

### 収益源の優先順位

| 順位 | 収益源 | 理由 |
|------|--------|------|
| 1位 | noteの単品有料記事 | 初手として検証しやすい。手数料も低い |
| 2位 | noteメンバーシップ | 最有力だが「毎月払う価値」が固まってから。手数料10% |
| 3位 | YouTube広告収益 | YPP到達が遠いため当面は補助線 |
| 4位 | アフィリエイト等 | 法務リスクを考慮し低優先のまま |

### note有料化の方針
- **課金の初手はメンバーシップではなく単品有料の小テスト**
- 500〜1,000人前後で「低単価の単品有料」を試す（5,000人を待たない）
- 無料記事で信頼を積みつつ、反応が良いテーマから単品有料を切る
- 構成: 無料（通常の安心記事）→ 単品有料（体系化したまとめ）→ メンバーシップ（継続価値が固まってから）
- 価格: 単体有料記事 300〜500円 / 有料マガジン 500〜980円 / メンバーシップ入口 500円/月
- 有料に向く内容:
  - **暴落時の心の整え方まとめ**（読み返し用）
  - **比較で揺れたときの整理まとめ**
  - **長期投資メンタルの再読用ハンドブック**
  - 無料記事の深掘り版
- note手数料: メンバーシップ10% / 定期購読マガジン20%（メンバーシップが有利だが運用難度は別）

### コミュニティ型収益
- **noteメンバーシップが最有力**（文章ベース・静かな継続・アーカイブ性がチャンネルに合う）
- ただし初手ではない。単品有料で「何が売れるか」を検証してからメンバーシップに移行
- 「暴落の夜に読む限定記事」「月1回の振り返り」「限定Q&A」
- YouTubeメンバーシップは2位（視聴者とのやり取りより"読む・聴く"体験が主のため）
- Discordは3位（雑談が増えるとチャンネルの静けさが崩れる）

### 投資ジャンルの法的リスク（重要）
- 金融庁は、有価証券等の価値分析に基づく投資判断への助言に当たる場合、投資助言・代理業の登録が必要と整理
- 無料の制度説明・過去データの一般論・感情整理・継続の考え方は比較的安全側
- **有料化すると一気に危うくなる線がある**
- 絶対に避けること:
  - 「今どれを買うべきか」「この商品とどちらを選ぶべきか」
  - 「いつ買う・いつ売る」のタイミング助言
  - 「あなたならこの配分」のような個別提案
  - 個別質問に対して具体商品を答えるQ&A
- 有料記事・メンバーシップで扱ってよい内容:
  - 暴落時の気持ちの整え方
  - 比較疲れの整理
  - 長期投資を続けるための環境設計
  - 制度の読み方・過去データの一般的な見方
- **有料Q&Aは「制度の仕組み」「記事テーマの補足」に限定し、個別商品の選定相談は受けない**
- **本格課金の前に、金融規制に強い専門家へ1回確認する**

### 証券口座アフィリエイト
- **優先度は低い**。法務リスクが高い
- 個別具体の勧誘、比較優劣の断定、特定商品の勧奨は避ける
- やるとしても制度説明は一般論に限定、免責を明記
