## Shorts台本構成（5シーン・12〜15秒）

感情曲線: **痛みワードで掴む → 共感 → データで安心 → 断言・希望 → CTA**

| # | ロール | 秒数 | 目的 | slide_text |
|---|---|---|---|---|
| 1 | hook | 2秒 | 痛みワード1語で掴む | 6文字以内（暴落。含み損。売りたい。） |
| 2 | empathy | 3秒 | 共感 + 挨拶 | ナレーション要点 |
| 3 | data | 4秒 | 具体的数字で安心材料 | 22文字以内 |
| 4 | resolve | 3秒 | 断言・希望 | 5〜7文字（時間が武器。退場＝負け。） |
| 5 | closing | 2秒 | CTA | 「明日もガチホしたい人はチャンネル登録」（固定） |

### 最強台本テンプレ（ChatGPT分析）

「7秒で意味が完結する構造」が最も再生が伸びる。言葉の長さを固定すると安定。

```
hook（1秒）: 1〜2語     例: 含み損 / 売りたい / 暴落
empathy（2秒）: 共感    例: あなただけじゃない / まだ残ってますか
data（4秒）: 数字       例: 20年続けた人、元本割れゼロ
resolve（2秒）: 結論    例: 時間が最大の武器 / 退場したら終わり
closing（2秒）: 質問    例: 続けてますか / 同じ人いますか
```

完成例（約12秒）:
```
含み損。あなただけじゃない、まだ残ってますか。
20年続けた人、元本割れゼロ。時間が最大の武器。続けてますか。
```

### hookの強さランキング
強い: ① 含み損 ② 暴落 ③ 売りたい ④ 不安 ⑤ 退場
弱い: 複利 / 分散投資 / 資産形成（感情が弱い）

### closingの改善案（コメント誘発型）
現在の固定CTA「チャンネル登録」→ 質問型に変更検討
- 「続けてますか」「同じ人いますか」→ コメント率UP

## 投稿順ルール

1. **hookが強いものを優先**（含み損 > 暴落 > 売りたい > 不安 > 焦り）
2. **同ジャンル連続を避ける**（ラウンドロビンでhook分散済み）
3. 理想の並び: 暴落→含み損→積立→売りたい→時間→...

## アルゴリズム攻略5つ

1. **最初の1秒** — 痛みワード1語で視聴継続率を上げる
2. **12〜15秒** — 完走率が高い＝アルゴリズム評価UP
3. **ループ再生** — closing→hookが自然に繋がると視聴回数増（今後対応）
4. **1日2本投稿** — Shorts最適頻度
5. **同ジャンル連続を避ける** — テーマ分散で評価維持

## 映像・演出方針

### スライド画像
- ダークトーン背景 + シルエット人物のフラットイラスト（プリメイド12枚）
- テキストは中央配置、影付き太字（ヒラギノ角ゴシック W9, 120pt）
- ロール別オーバーレイカラー（hook:赤系, data:青系, resolve:緑系）

### 字幕
- Pillowでスライド画像に焼き込み（下から420px位置、YouTubeUI回避）
- 白文字 + 黒縁（ストローク3px）
- 15文字/行で改行

### AI音声
- ElevenLabs eleven_multilingual_v2
- speed=1.15（思想系の落ち着いたテンポ）
- stability=0.4, similarity_boost=0.75, style=0.2
- 投資用語の読み替え辞書あり（387エントリ）

### BGM
- assets/bgm_ambient.m4a（静かなアンビエント）
- 音量: BGM_VOLUME=0.08（ナレーションの邪魔をしない）
- ループ再生、フェードイン1秒

### resolve前の間
- 0.7秒の無音クリップ挿入（断言前の「間」演出）

## 収益化戦略（2026-03-10 ChatGPT分析）

### 基本方針
- **広告だけで回収する型ではなく、信頼を積み上げて複数の収益源を重ねる型**
- Shorts広告は構造的に薄い（Shorts Feed Ads、クリエイター取り分45%、RPMも低い）
- 三層設計: **Shorts = 新規流入** / **長尺 = 信頼形成+広告収益** / **note = 深掘り+課金**

### repetitious content リスク（重要）
- YouTube は 2025年7月に「repetitious content」を「inauthentic content」と明確化
- 反復的・量産的に見える作りは収益化審査で不利になりえる
- AI音声・自動化自体が即NGではないが、独自性と一貫した人格が重要
- 対策: テーマ重複回避（dedupe_check.py）、音声・構造の多様性を意識

### 段階別ロードマップ

| 段階 | やること | 収益源 |
|------|---------|--------|
| 〜100人 | 世界観固定。長尺週1で試す。note無料記事を増やす | なし |
| 〜1,000人 | YPP到達。長尺週1継続。外部の受け皿整備 | YouTube広告（少額） |
| 〜5,000人 | 収益源2本目。note有料記事。YouTube Shopping affiliate | 長尺広告 + note有料 |
| 〜1万人 | コミュニティ型月額課金を検討 | 上記 + メンバーシップ |
| 〜10万人 | 自社商品・提携企画・書籍 | 広告 + note + メンバーシップ + 商品 |

### note有料化の方針
- **今は無料記事で信用を作る段階**。全面有料化はしない
- 無料20〜30本蓄積後に有料化開始
- 構成: 無料（通常の安心記事）/ 有料（体系化したまとめ）/ 月額（限定Q&A・限定連載）
- 価格: 単体有料記事 300〜500円 / 有料マガジン 500〜980円 / メンバーシップ入口 500円/月
- 有料に向く内容: 暴落時の心の整え方まとめ / 場面別Q&A集 / 長期投資メンタルの再読用ハンドブック

### コミュニティ型収益
- **noteメンバーシップが最有力**（文章ベース・静かな継続・アーカイブ性がチャンネルに合う）
- 「暴落の夜に読む限定記事」「月1回の振り返り」「限定Q&A」
- YouTubeメンバーシップは2位（視聴者とのやり取りより"読む・聴く"体験が主のため）
- Discordは3位（雑談が増えるとチャンネルの静けさが崩れる）

### 証券口座アフィリエイト
- **優先度は低い**。法務リスクが高い
- 個別具体の勧誘、比較優劣の断定、特定商品の勧奨は避ける
- やるとしても制度説明は一般論に限定、免責を明記

### このチャンネル独自の収益化アイデア
1. **暴落時の読み直し用ミニブック**（PDF/note有料マガジン） — 含み損の夜/利確したい夜/比較して焦る夜ごとに1ページ
