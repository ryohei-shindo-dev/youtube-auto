| 項目 | 内容 |
|---|---|
| チャンネル名 | ガチホのモチベ |
| サブタイトル | @長期投資を続けたい人のチャンネル |

## Shorts台本構成（5シーン）

感情曲線: **不安ワードで掴む → 共感 → データで安心 → 断言・希望 → CTA**

| # | ロール | 秒数 | 目的 | スライドテキスト |
|---|---|---|---|---|
| 1 | hook | 3秒 | 不安ワードで掴む（最初の1.5秒が勝負） | 不安を刺す短文 |
| 2 | empathy | 4秒 | 共感 + 挨拶「今日もガチホしてますか？」 | ナレーションの要点 |
| 3 | data | 5秒 | 具体的な数字・データで安心材料 | ナレーションの要点 |
| 4 | resolve | 4秒 | 断言・希望 + 結論フレーズ（複数パターン） | 断言フレーズ |
| 5 | closing | 2秒 | CTA固定フレーズ | 「明日もガチホしたい人はチャンネル登録」（固定） |

※ 合計 17秒目標（16〜18秒）
※ hookを挨拶より前に置くことで最初の1.5秒で離脱を防ぐ
※ 結論フレーズは5パターンからランダム選択

## 映像・演出方針

### スライド画像（プリメイド12枚セット）
- ダークトーン背景 + シルエット人物のフラットイラスト
- テーマ×ロールで使い分け（例: hookは不安系画像、resolveは希望系画像）
- テキストは中央配置、影付き太字（ヒラギノ角ゴシック W9, 120pt）

### hookスライド → 不安・共感の画像（下落チャート、スマホを見て焦る人）
### dataスライド → データ系画像（長期チャート、成長グラフ）
### resolveスライド → 希望系画像（喜ぶ人、日の出）

### サムネイル
- 曜日ごとに色テンプレ固定（月:緑, 火:金, 水:赤, 木:青, 金:橙）
- アクセントバー + テーマラベル + タイトル + チャンネルバッジ

### AI音声
- ElevenLabs eleven_multilingual_v2
- speed=1.15（思想系の落ち着いたテンポ）
- stability=0.4, similarity_boost=0.75, style=0.2
- 投資用語の読み替え辞書あり（元本→がんぽん等）

## コンテンツポリシー

### 絶対にやらないこと
- 個別株の推奨
- 「今が買い時」等の投資助言
- 短期トレード解説
- 煽り・誇張・サムネ詐欺

### 必ずやること
- 全動画の結論を「やっぱり、長期投資しかないですね」に収束
- 概要欄に「※投資助言ではありません」の免責事項
- 視聴後に「長期投資を続けてよかった」と思える内容

## スプレッドシート管理

- 100本のトピック（Shorts 80本 + 通常 20本）を投稿管理シートで管理
- ステータス: 未生成 → 生成済み → 公開済み
- テーマ別に分類（Shorts/メリット, Shorts/格言 等）

## 技術パイプライン

```

  → [台本生成] Claude API (claude-sonnet-4-6)
  → [音声生成] ElevenLabs REST API
```

## ChatGPTレビュー指摘（累計2回分）の要約

### 評価されている点
- 雰囲気・トーンは良い（暗め、静か、落ち着き → 思想系チャンネルの空気）
- AI動画としての完成度は高い（映像として成立している）
- 方向性はブルーオーシャン（煽り系が多い投資YouTubeで差別化できる）

### 繰り返し指摘されている問題
1. **冒頭フックが弱い** — 最初の2〜3秒で「不安ワード」が必要。今は静かすぎて離脱される
2. **感情のピークがない** — 感情曲線が平坦。教育動画の構造になっている
3. **断言フレーズがない** — 記憶に残る一文がない。「暴落は味方だ」「退場した人だけが損をする」レベルの強い言い切りが必要
4. **視聴者の痛みに触れていない** — 含み損・SNS焦り・積立やめたい等の「自分ごと化」が弱い
5. **1動画1メッセージになっていない** — Shortsは一撃。複数の話を詰め込まない
6. **AI音声の強弱不足** — 断言部分は速度を落とす・間を取る等のメリハリが必要

### ChatGPT戦略レビュー（3回目: 戦略ファイルに対する評価）

**戦略完成度: 8/10**（YouTube最適化を入れれば9.5まで行ける）

#### 主要指摘
1. **opening 2秒 → hookを先に出すべき** — 挨拶から始まる動画は弱い。最初の1.5秒で「不安ワード」→ その後に挨拶の順にする
2. **結論フレーズ固定は飽きる** — 毎回同じ結論だと慣れる。3〜5パターン用意してローテーション
3. **Shorts 25秒は長い** — 18秒前後が最も維持率が高い。15〜18秒を目標にする
4. **BGMなしは危険** — 思想チャンネルでも完全無音は維持率を落とす。静かなピアノ -28〜-32 LUFS
5. **火曜「格言」テーマが弱い** — 格言単体はTikTok寄り。ストーリー化が必要（例:「バフェットが20代でやらなかったこと」）
6. **哲学チャンネルになりきれていない** — 記憶に残る名言・断言フレーズをもっと増やすべき
7. **動画を2フォーマットに** — shorts_motivation / shorts_data でアルゴリズムに多様性を見せる
8. **シリーズ化で強化** — ガチホ哲学 / 暴落の真実 / 長期投資の歴史 / ガチホ格言
